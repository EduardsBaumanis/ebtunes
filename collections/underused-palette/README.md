# Underused Palette

A 40-track Strudel album built from the catalog underuse list: rare sample
tokens, alphabet and number fragments, indexed drum variants, low-count
noise colors, and unusually high or low pitch spellings.

Each track deliberately foregrounds at least one low-count sound token and
one low-count explicit note or register color from `docs/SONG_GENERATION_CATALOG.md`.
